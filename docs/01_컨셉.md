# 온MINIMUM — 브랜드 컨셉

## 브랜드명

**온MINIMUM** / **ONMinimum**

---

## 네이밍 의미

| 단어 | 의미 |
|------|------|
| **온** (韓) | 온전하다 — 전체, 완전함, 결핍 없음 |
| **On** (英) | 켜진 상태, 활성화, 존재의 시작 |
| **MINIMUM** | 최소, 본질만 남긴 것 |

"온"은 중의적 표현으로, 한국어의 완전함과 영어의 On(켜짐)이 동시에 읽힌다.
→ **완전히 켜진 최소** / **온전한 최소**

---

## 브랜드 철학

> "더 이상 뺄 것이 없을 때 완성된다."

불필요한 것을 걷어내고 본질만을 남겼을 때, 비로소 완전함이 드러난다.
최소한의 것으로 최대한의 의미를 만드는 것.

### 3가지 핵심 가치

1. **온전함 (Wholeness)** — 부분이 아닌 전체, 표면이 아닌 본질
2. **최소함 (Minimalism)** — 더 이상 뺄 것이 없는 순수한 형태
3. **켜짐 (Activation)** — 잠든 가능성을 깨우는 On의 상태

---

## 로고 마크

SVG 기반 기하학적 마크. 세 겹의 요소로 구성.

```
큰 원 (실선)   = 온전함 — 결핍 없는 완전한 전체
점선 링        = MINIMUM — 최소의 경계
중심 점        = On — 켜짐, 빛, 활성화의 기원
십자 기준선    = 중심축, 방향성
```
